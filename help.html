How IPTV Detective works — simple explanation

This site helps you check an IPTV provider's channel list and compare it with other public submissions to find identical or very similar packages. It's useful for spotting resellers (different shops selling the same package) and comparing prices.

How to use it

  1. Select the Check Provider tab and enter the provider information: provider name, provider website URL, and the yearly price.
  2. Choose the provider's .m3u file (the playlist file that lists channels). If the file is large, parsing may take a minute.
  3. Submit the form; the site will compare the channels and groups in your file to other submissions and show:
    • How many channels and groups the file has.
    • A hash (a short fingerprint) for the file so identical files can be found.
    • A similarity/compare result showing matching providers and a visual match score.

Privacy and what we send

We do not upload the full M3U file to the server for most checks — only small counts and fingerprints are used to compare packages. This keeps uploads fast and avoids storing full playlists publicly. If you choose to submit a provider, that submission may be stored and displayed.

Why this helps

Resellers often copy the same channel list and sell it under different names. By comparing the channel list fingerprints and group lists, the site can detect when different providers are actually the same package, helping you find better prices or spot duplicated offers.
